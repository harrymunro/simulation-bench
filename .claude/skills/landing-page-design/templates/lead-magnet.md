# Template: Lead Magnet Landing Page

## Purpose
Capture email addresses in exchange for a free resource (ebook, guide, checklist, template).

## Best for
- Free ebook downloads
- Guides and cheat sheets
- Templates and starter kits
- Checklists and frameworks

## Key Principle
**High value, low friction.** The page should be short and focused. Don't oversell a free resource.

## Recommended Section Order

1. Hero (what they get + email capture)
2. What's Inside (quick preview of content)
3. Who It's For (qualify the reader)
4. Author credibility (brief)
5. Secondary CTA (repeat form)

## What NOT to Include

Lead magnet pages should be **short and focused**. Avoid:

- ❌ Long sales copy
- ❌ Multiple CTAs or offers
- ❌ Extensive testimonials
- ❌ Detailed author bios
- ❌ FAQ sections (if they have questions, they won't download)
- ❌ Pricing comparisons
- ❌ Feature lists beyond what's inside

## Mobile Considerations

- Hero: Stack mockup above form
- Form: Full-width inputs
- Download button: Full-width, prominent
- Keep total scroll minimal

## Form Best Practices

**Minimum fields = maximum conversions**

```
┌────────────────────────────────────────┐
│  Ideal: Just email                     │
│  ┌──────────────────────────────────┐  │
│  │  Email address                   │  │
│  └──────────────────────────────────┘  │
│  ┌──────────────────────────────────┐  │
│  │  Send me the guide →             │  │
│  └──────────────────────────────────┘  │
└────────────────────────────────────────┘

┌────────────────────────────────────────┐
│  Acceptable: Email + first name        │
│  ┌──────────────────────────────────┐  │
│  │  First name                      │  │
│  └──────────────────────────────────┘  │
│  ┌──────────────────────────────────┐  │
│  │  Email address                   │  │
│  └──────────────────────────────────┘  │
│  ┌──────────────────────────────────┐  │
│  │  Get instant access →            │  │
│  └──────────────────────────────────┘  │
└────────────────────────────────────────┘
```

> **Avoid**: Asking for job title, company, phone number, or
> anything not essential. Every extra field loses conversions.

## Button Copy That Works

Instead of "Submit" or "Download", use:

- "Send me the guide"
- "Get instant access"
- "Download now"
- "Get your free copy"
- "Yes, I want this"

- Core SimPy patterns with code examples
