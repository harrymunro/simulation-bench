# Template: Product Launch Landing Page

## Purpose
Convert interested visitors into customers for a course, bootcamp, or paid product.

## Best for
- Course launches (bootcamps, masterclasses)
- Paid products and services
- High-ticket offerings requiring trust-building

## Recommended Section Order

1. Hero (with clear value proposition)
2. Problem (agitate the pain)
3. Solution (introduce the product)
4. Benefits (outcomes they'll achieve)
5. Features (what's included)
6. Social Proof (testimonials, logos)
7. Instructor/Creator (build trust)
8. FAQ (handle objections)
9. Final CTA (with urgency)

## Mobile Considerations

- Hero: Stack image below text
- Benefits: Single column, stacked
- Features: Image above text for each
- Testimonials: Single column, swipeable
- CTA buttons: Full width on mobile
