# Business Context

## Overview

Teaching engineers how to build discrete-event simulation models using Python and SimPy. Founded by Harry, a British engineer with a decade of industry experience.

## Core Value Proposition

**For engineers frustrated with expensive, inflexible commercial simulation software:**
- SimPy is free and open source (no licence fees)
- Python integrates with your existing data workflow
- Custom solutions instead of vendor-imposed limitations
- You own your code and your models

**The transformation we offer:**
"Go from beginner to expert" - become the go-to simulation person in your organisation.
